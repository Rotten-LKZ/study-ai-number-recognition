项目背景

由于该课程是基本的机器学习入门课程，并且着重于 Python 开发，所以选择了简单的手写数字识别作为练习项目，理解项目实际开发的流程并实际跑通。这是一个基于深度学习的手写数字识别系统，旨在提供一个简单易用的界面，让用户能够上传手写数字图片并获得识别结果。该项目使用了卷积神经网络模型进行训练，能够准确识别0-9的手写数字。

使用步骤

  1. uv sync 安装依赖
  2. uv run train_mnist.py 首次运行训练模型
  3. uv run main.py 启动应用
  4. 注册账号之后根据注册的账号密码登录
  5. 访问首页，点击“选择图片”按钮上传一张包含手写数字的图片。
  6. 点击“识别”按钮，系统将处理图片并显示识别结果。
  7. 在结果页面，您可以查看识别的数字及其对应的置信度。
  8. 如果需要重新识别，可以点击“重新识别”按钮返回首页。

开发环境

前端使用原生 HTML、CSS 和 JavaScript 开发，后端使用 Python 和 Flask 框架。使用 VSCode 进行开发，并采用 Claude Code 作为辅助工具。

依赖清单

  • Flask - Web 框架
  • SQLAlchemy - 数据库 ORM
  • PyTorch - 深度学习框架
  • Bootstrap - 前端 UI 框架
  • SweetAlert2 - 弹窗库
  • QRCode.js - 生成二维码

模型数据来源

MNIST 手写数字数据集
